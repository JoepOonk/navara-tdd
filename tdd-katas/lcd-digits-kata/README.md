# About the LCD Digits Kata

The goal of this kata is to convert any arbitrary (integer) number
into a string representation like so:

```
 _     _  _     _  _  _  _  _ 
| |  | _| _||_||_ |_   ||_||_|
|_|  ||_  _|  | _||_|  ||_| _|
```

For example, the number 318 must be rendered as

```
 _     _ 
 _|  ||_|
 _|  ||_|
```
